# 🌐 OAuth Authentication

Today's primary focus was **OAuth Authentication**.

OAuth is an authorization framework that allows users to log in to third-party applications without sharing their passwords directly.

Example:

- Login with Google
- Login with GitHub
- Login with Microsoft

Instead of providing your password to every application, OAuth securely authorizes access through a trusted identity provider.
