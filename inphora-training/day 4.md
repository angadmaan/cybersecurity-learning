# Types of Hackers

## White Hat Hacker
Ethical hackers who help organizations find and fix security vulnerabilities legally.

## Black Hat Hacker
Hackers who gain unauthorized access to systems for malicious purposes.

## Grey Hat Hacker
Hackers who may access systems without permission but usually do not have malicious intentions.

# DNS (Domain Name System)

DNS translates domain names into IP addresses.

Example:
google.com → 142.250.190.14

Goal:
Allow users to access websites using names instead of IP addresses.

---

# Ports

Ports are communication endpoints used by applications and services.

### Common Ports

- HTTP → Port 80

- HTTPS → Port 443

- SMTP (Email Sending) → Port 25

- POP3 (Email Receiving) → Port 110

- FTP → Port 20
 
- DNS → Port 53

---

# Protocols

Protocols are rules that define how devices communicate over a network.

Examples:
- HTTP
- HTTPS
- FTP
- SMTP
- DNS

---

# FTP (File Transfer Protocol)

FTP is used to transfer files between computers over a network.

Goal:
Upload and download files between client and server.

Port:
- FTP Control → 21
- FTP Data Transfer → 20

---
