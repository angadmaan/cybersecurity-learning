## Overview

This repository contains my personal learning notes and documentation from the **Play It Safe: Manage Security Risks** course, the second course in the **Google Cybersecurity Professional Certificate** offered by **Google** on **Coursera**.

This course builds upon the cybersecurity fundamentals introduced in the first course and focuses on understanding organizational security, managing cyber risks, applying security frameworks, using security controls, and responding to security incidents through structured playbooks.

Instead of simply completing the course, I documented each module to strengthen my understanding and create a structured reference that I can revisit throughout my cybersecurity journey.

## Learning Outcome

After completing this course, I developed a stronger understanding of how organizations identify, assess, and manage cybersecurity risks. I learned how security frameworks and controls help reduce threats, why security documentation is important, and how structured incident response processes enable security teams to react effectively during cyber incidents.

This course also introduced practical concepts related to security operations, organizational policies, compliance requirements, and playbooks that are commonly used by Security Analysts in real-world environments.
