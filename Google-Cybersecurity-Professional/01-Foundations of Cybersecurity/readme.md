## Overview

This repository contains my personal learning notes and documentation from the **Google Foundations of Cybersecurity** course offered by **Google** on **Coursera**.

Instead of simply completing the course and earning a certificate, I documented each module to reinforce my understanding and create a structured knowledge base that I can revisit in the future. These notes summarize key cybersecurity concepts, frameworks, tools, and best practices introduced throughout the course.

The goal of this repository is to track my cybersecurity learning journey while building a public portfolio of well-organized study material.
