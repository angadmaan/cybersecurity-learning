# Linux File Permission Commands

## chmod
Used to change file permission.

Example:
chmod 755 file.txt

Example:
chmod +x script.sh

---

## chown
Used to change the owner of a file.

Example:
chown user file.txt

Example:
chown user:group file.txt

---

## chgrp
Used to change the group ownership of a file.

Example:
chgrp developers file.txt

---

## ls -l
Shows file permissions, owner, and group details.

Example:
ls -l

Output example:
-rwxr-xr-- 1 user group file.txt

---

## Permission Types

r – read permission  
w – write permission  
x – execute permission
