# MAC Address

## Introduction

A MAC (Media Access Control) Address is a unique physical identifier assigned to a network interface card (NIC).

Unlike IP addresses, MAC addresses are generally assigned by the manufacturer and remain fixed for the hardware.

---

## Structure of a MAC Address

Example:

00:1A:2B:3C:4D:5E

Characteristics:

- 48 bits long
- Written in hexadecimal format
- Unique for every network interface

---

## Purpose of MAC Addresses

MAC addresses are used to:

- Identify devices on a local network
- Enable communication within LANs
- Support data delivery at the Data Link Layer

---

## MAC Address vs IP Address

| MAC Address | IP Address |
|------------|------------|
| Physical Address | Logical Address |
| Assigned by Manufacturer | Assigned by Network |
| Usually Permanent | Can Change |
| Used in Local Network | Used Across Networks |

---

## How MAC Addresses Work

When devices communicate within a local network:

1. Device identifies destination IP.
2. ARP is used to find the MAC address.
3. Data is delivered using the MAC address.

---

## MAC Address Spoofing

Attackers may change their MAC address to:

- Bypass restrictions
- Hide identity
- Impersonate another device

This technique is known as MAC Spoofing.

---

## Importance in Cybersecurity

MAC addresses help security professionals:

- Identify devices
- Monitor network activity
- Detect unauthorized systems
- Investigate incidents

---

## Key Learning

MAC addresses uniquely identify network hardware and play a critical role in communication within local networks.
