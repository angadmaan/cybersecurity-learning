# Wireless Networking

## Introduction

Wireless networking allows devices to communicate without physical cables using radio signals.

The most common wireless technology is Wi-Fi.

---

## Advantages

- Mobility
- Easy installation
- Flexible connectivity
- Reduced cabling

---

## Components

### Wireless Router

Provides internet access and Wi-Fi connectivity.

### Access Point (AP)

Extends wireless network coverage.

### Wireless Client

Any device connected to Wi-Fi.

Examples:

- Smartphones
- Laptops
- Tablets

---

## Wi-Fi Standards

Common standards:

- 802.11n
- 802.11ac
- 802.11ax (Wi-Fi 6)

Each generation improves speed and performance.

---

## Wireless Security

### WPA2

Widely used wireless security protocol.

### WPA3

Latest wireless security standard with improved protection.

---

## Common Wireless Threats

### Rogue Access Point

Unauthorized wireless access point.

### Evil Twin Attack

Fake Wi-Fi network designed to steal information.

### Packet Sniffing

Capturing wireless traffic.

---

## Best Practices

- Use WPA3 or WPA2
- Create strong passwords
- Disable unused features
- Update router firmware

---

## Importance in Cybersecurity

Wireless networks are common attack targets and must be secured properly.

---

## Key Learning

Wireless networking provides convenient connectivity but requires proper security measures to protect users and data.
