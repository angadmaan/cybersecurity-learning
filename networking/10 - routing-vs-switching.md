# Routing vs Switching

## Introduction

Routing and switching are two fundamental networking functions that enable devices to communicate.

Switches connect devices within the same network, while routers connect different networks together.

---

## Switching

A switch operates mainly at Layer 2 (Data Link Layer) of the OSI model.

It uses MAC addresses to forward data between devices on the same Local Area Network (LAN).

### Functions

- Connects devices within a LAN
- Learns MAC addresses
- Reduces unnecessary traffic
- Improves communication efficiency

### Example

A switch allows computers, printers, and servers in an office network to communicate with each other.

---

## Routing

A router operates mainly at Layer 3 (Network Layer).

It uses IP addresses to move data between different networks.

### Functions

- Connects different networks
- Chooses the best path for data
- Provides internet access
- Separates broadcast domains

### Example

A home router connects your local network to the internet.

---

## Switch vs Router

| Switch | Router |
|----------|----------|
| Uses MAC Address | Uses IP Address |
| Works inside LAN | Connects different networks |
| Layer 2 Device | Layer 3 Device |
| Faster local communication | Routes traffic between networks |

---

## Key Learning

Switches handle communication inside a network, while routers connect networks together and direct traffic between them.
