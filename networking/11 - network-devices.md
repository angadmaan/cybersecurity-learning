# Network Devices

## Introduction

Network devices enable communication between computers and networks.

Each device performs a specific role in managing and forwarding data.

---

## Hub

A hub broadcasts incoming data to all connected devices.

Characteristics:

- Simple device
- No intelligence
- Creates unnecessary traffic

Today hubs are rarely used.

---

## Switch

A switch forwards data only to the intended destination.

Characteristics:

- Uses MAC addresses
- Improves performance
- Commonly used in LANs

---

## Router

A router connects multiple networks.

Characteristics:

- Uses IP addresses
- Connects LAN to WAN
- Provides internet access

---

## Modem

A modem converts signals between the ISP and the local network.

Functions:

- Connects home networks to the internet
- Works with routers

---

## Access Point (AP)

Provides wireless connectivity.

Functions:

- Extends Wi-Fi coverage
- Allows wireless devices to join the network

---

## Firewall

A security device that filters network traffic.

Functions:

- Allow authorized traffic
- Block malicious traffic
- Protect networks

---

## Gateway

Acts as an entry and exit point between networks.

Functions:

- Enables communication between different systems
- Performs protocol conversion if necessary

---

## Key Learning

Switches, routers, firewalls, access points, and other devices work together to build secure and reliable networks.
