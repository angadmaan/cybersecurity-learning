# IP Addressing

## Introduction

An IP (Internet Protocol) Address is a unique numerical identifier assigned to a device connected to a network. It allows devices to communicate and exchange data over local networks and the internet.

Just as a postal address helps deliver letters to the correct location, an IP address helps data reach the correct device.

---

## Purpose of IP Addressing

IP addresses are used to:

- Identify devices on a network
- Enable communication between devices
- Route data across networks
- Support internet connectivity

Without IP addresses, devices would not know where to send or receive information.

---

## IPv4 Address

IPv4 is the most commonly used version of IP.

Example:

192.168.1.10

Characteristics:

- 32-bit address
- Consists of four octets
- Each octet ranges from 0 to 255

---

## IPv6 Address

IPv6 was introduced to overcome the shortage of IPv4 addresses.

Example:

2001:0db8:85a3::8a2e:0370:7334

Characteristics:

- 128-bit address
- Much larger address space
- Improved efficiency and security

---

## Public and Private IP Addresses

### Public IP Address

Assigned by an Internet Service Provider (ISP).

Used for communication over the internet.

Example:

103.25.50.100

---

### Private IP Address

Used inside local networks.

Examples:

- 192.168.x.x
- 10.x.x.x
- 172.16.x.x to 172.31.x.x

Private IP addresses are not directly accessible from the internet.

---

## Static and Dynamic IP Addresses

### Static IP

Remains unchanged.

Commonly used for:
- Servers
- Network devices

### Dynamic IP

Assigned automatically by DHCP.

Commonly used for:
- Home users
- Office workstations

---

## Importance in Cybersecurity

Understanding IP addressing helps in:

- Network troubleshooting
- Traffic monitoring
- Access control
- Threat investigation
- Log analysis

---

## Key Learning

IP addresses uniquely identify devices and make network communication possible. Both IPv4 and IPv6 play an important role in modern networking.
