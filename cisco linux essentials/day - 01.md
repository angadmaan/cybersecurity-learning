## Virtual Terminal

The Virtual Terminal provides a Command Line Interface (CLI) to practice Linux commands.

Example Prompt:

sysadmin@localhost:~$

The prompt displays:

- Current user (sysadmin)
- System name (localhost)
- Current directory (~)

---

## Root User

Some administrative tasks require root privileges.

Command:

su

After entering the correct password, the prompt changes to:

root@localhost:~#

The # symbol indicates the root user.

---

## Current Directory

The prompt also shows the current working directory.

Example:

cd Documents

Prompt changes to:

sysadmin@localhost:~/Documents$

---

## Useful Command

Display current user:

whoami

Example Output:

sysadmin

---

## Reset Environment

The virtual environment may reset automatically or manually.

Effects:

- Created files may be removed
- Environment settings may be lost
- Aliases and variables may be reset
