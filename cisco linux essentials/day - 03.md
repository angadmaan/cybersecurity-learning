# Day 3

## Operating System (OS)

An Operating System is software that manages:

- Hardware resources
- Software resources
- Running applications
- User interaction

Examples:
- Windows
- macOS
- Linux

---

## Roles of a Computer

### Desktop

Used for:
- Web browsing
- Office work
- Multimedia

Usually uses:
- GUI (Graphical User Interface)

---

### Server

Used for:
- Hosting websites
- File sharing
- Databases
- Network services

Usually uses:
- CLI (Command Line Interface)

---

## Factors in Choosing an Operating System

### Role
Desktop or Server

### Function
Tasks and applications required

### Life Cycle
Support and update period

### Stability
Reliable software with fewer bugs

### Compatibility
Ability to run existing applications

### Cost
Licensing and maintenance expenses

### Interface
GUI or CLI

---

## Major Operating Systems

### Microsoft Windows

Features:
- Proprietary
- User-friendly GUI
- Strong backward compatibility

Examples:
- Windows 10
- Windows 11
- Windows Server

---

### Apple macOS

Features:
- UNIX-certified
- Easy to use
- Popular among creative professionals

Based on:
- BSD UNIX

---

### Linux

Features:
- Open Source
- Flexible
- Highly customizable

Obtained through:
- Linux Distributions

---

## Linux Distribution

A Linux distribution contains:

- Linux Kernel
- GNU Tools
- Applications
- Package Manager

Examples:

- Ubuntu
- Debian
- Fedora
- Red Hat Enterprise Linux (RHEL)
- Kali Linux
- Linux Mint
- openSUSE

---

## Types of Linux Distributions

### Enterprise Distributions

Focus:
- Stability
- Long-term support

Examples:
- RHEL
- SUSE Linux Enterprise
- Ubuntu LTS

---

### Enthusiast Distributions

Focus:
- Latest features
- Fast updates

Examples:
- Fedora
- openSUSE Tumbleweed

---

## Android

Android uses the Linux kernel but is different from traditional desktop Linux distributions.

Used in:
- Smartphones
- Tablets

---

## Embedded Systems

Embedded systems are devices designed for specific tasks.

Examples:

- Smart TVs
- Routers
- IoT Devices
- Raspberry Pi
- DVR Systems

Many embedded systems use Linux because it is flexible and lightweight.

---

## Key Learning

Linux is a flexible, open-source operating system available through many distributions. It can be used on desktops, servers, mobile devices, and embedded systems, while offering both GUI and CLI interfaces.
