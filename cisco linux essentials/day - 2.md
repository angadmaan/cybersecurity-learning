# Day 2

## What is Linux?

Linux is the kernel of an operating system. The kernel manages hardware resources and controls how the system operates.

When people say "Linux", they usually mean GNU/Linux, which combines:

- Linux Kernel
- GNU Tools and Utilities

Together they form a complete operating system.

---

## UNIX and Linux

Linux was inspired by UNIX, an operating system developed in the 1970s.

Characteristics of UNIX:

- Written in C language
- Portable across different hardware
- Modular design
- Influenced many modern operating systems

Linux is considered UNIX-like but is not officially UNIX.

---

## History of Linux

Linux was created in 1991 by Linus Torvalds while studying at the University of Helsinki.

His goal was to create a free and open-source operating system kernel.

---

## GNU Project

The GNU Project was started by Richard Stallman in 1983.

GNU contributed:

- Compilers
- Editors
- System Utilities
- User Interfaces

Linux and GNU tools together created the GNU/Linux operating system.

---

## Open Source Software

Open source software provides access to source code.

Benefits:

- Transparency
- Customization
- Community Development
- Faster Innovation

Linux is one of the most successful open-source projects.

---

## Linux Distributions

A Linux Distribution (Distro) combines:

- Linux Kernel
- GNU Tools
- Applications
- Package Manager

Examples:

- Ubuntu
- Debian
- Red Hat Enterprise Linux (RHEL)
- Kali Linux
- Fedora

---

## User Interfaces

### GUI (Graphical User Interface)

Uses windows, icons, menus, and mouse interaction.

Examples:

- Web Browsers
- File Managers
- Office Applications

---

### CLI (Command Line Interface)

A text-based interface where users enter commands using a keyboard.

Benefits:

- Faster execution
- Precise control
- Automation through scripting
- Works across most Linux distributions

---

## Key Learning

Linux is a free and open-source UNIX-like operating system that combines the Linux kernel with GNU tools. Users can interact with Linux using either a GUI or CLI, with the CLI being especially powerful for system administration and cybersecurity tasks.
