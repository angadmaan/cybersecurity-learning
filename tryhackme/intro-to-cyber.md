# TryHackMe – Intro to Cyber Security

## Offensive Security
Offensive security focuses on attacking systems to find vulnerabilities before malicious hackers do.  
Professionals in this field simulate real-world attacks to identify weaknesses in networks, applications, and systems.

Common roles:
- Penetration Tester
- Ethical Hackers
- Red Team Analyst

Goal: To Discover and report security flaws so organizations can fix them.

---

## Defensive Security
Defensive security focuses on protecting systems and detecting attacks.  
Professionals monitor networks, analyze threats, and respond to security incidents to keep systems secure.

Common roles:
- Security Analyst
- Blue Team Analyst
- Security Engineer
- Incident Responder

Goal: Prevent attacks, detect suspicious activity, and minimize damage.

---

## Careers in Cybersecurity
Cybersecurity offers multiple career paths depending on interests and skills.

Examples of roles:
- Penetration Tester (Offensive)
- Security Analyst (Defensive)
- Security Engineer
- Threat Intelligence Analyst
- Digital Forensics Expert

To build a career in cybersecurity, strong fundamentals in networking, Linux, operating systems, and security concepts are essential.
