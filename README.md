## 📌 About

This repository is my working record while learning cybersecurity — networking
fundamentals, Linux administration, hands-on labs, and simulated real-world
security work. It's not a polished portfolio; it's the actual trail of notes,
scripts, and write-ups from the process. Expect rough edges, frequent commits,
and continuous updates.

I'm a B.Tech Computer Science student at Guru Nanak Dev University,
specializing in network security and building toward a cybersecurity role —
this repo is where that build-up actually happens.
